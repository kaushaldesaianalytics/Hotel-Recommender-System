Recommend.py is a implementation of getting the Collabrative filtering recommender. It finds the mostpopular hotels
in three categories as explained in report.

to run: Download test and train data from https://www.kaggle.com/c/expedia-hotel-recommendations/data in the same
folder as the script.
run: python recommnder.py
run: python kmodes-expedia.py 

Data Preprocessing and Feature Selection methods are done in MS Azure and the workspace is shared with Professor Anessi Bari.
